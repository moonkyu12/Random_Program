# Random_Program

PyInstaller build:

- Install build dependencies: `.\venv\Scripts\python.exe -m pip install -r requirements-build.txt`
- Build both outputs: `.\build_pyinstaller.ps1`
- Onefile output: `dist\School Random Program.exe`
- Onedir output: `dist\School Random Program Onedir\School Random Program.exe`

Both builds use `console=False`, so no console window is shown.
